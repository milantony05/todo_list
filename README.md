## Installation

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/milantony05/todo_list.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd todo_list
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
